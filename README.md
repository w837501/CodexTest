# Java Hello World

這是一個使用 Gradle 結構的最小 Java 網頁專案。

## 結構

```text
build.gradle
settings.gradle
src/main/java/Main.java
```

## 執行

如果你的環境已安裝 Gradle：

```bash
gradle run
```

看到這行代表伺服器已啟動：

```text
Server started at http://localhost:8080
```

接著打開瀏覽器進入：

```text
http://localhost:8080
```

頁面會顯示 `Hello World` 和一段簡單文字。
